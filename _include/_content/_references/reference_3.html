Battle for Metal

A network multiplayer game.

Information

Programming languages:
Java
Engine/Framework:
libGDX
Other used tools:
Eclipse, TiledMaps
Role:
Gamelogic

This game is a network multiplayer game developed during the GameDevWeek at Hochschule Trier. The goal of this game is to collect more metal on the map as your enemies. The goal can be archieved by shooting at your enemies with metal and be faster as them.

Under following link you will find the official wiki entry, there you can download the game or source-code. https://fsi.hochschule-trier.de/dev/null/Battle_for_Metal
