Volumentric Cloud Rendering for Real-Time Applications

This work is showing how to represent clouds in real-time applications as realistic as possible. The clouds are stored in 3D textures between two spheres, which imitate the atmosphere of a planet, processed with ray marching and displayed as a postprocessing effect on the screen. With the presented technique the clouds can be dynamically lit and viewed from any angle while keeping the memory budget low.
